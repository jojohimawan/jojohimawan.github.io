Membuat Student Site Dengan Layanan Hosting UPT Komputer PENS

Apa Itu Hosting UPT Komputer PENS?

Sebagai politeknik berbasis teknik terbaik di Indonesia, tentu saja PENS memiliki segudang fasilitas untuk menunjang kegiatan perkuliahan. Terlebih dalam fasilitas teknologi. Salah satu dari fasilitas tersebut adalah layanan hosting website dan cloud storage bernama FTP PENS.

Pada dasarnya Web FTP PENS adalah layanan khusus civitas akademika PENS guna menunjang perkuliahan di bidang IT. Dengan menggunakan fasilitas ini, mahasiswa aktif dapat membuat personal website mereka go live. Selain itu, mahasiswa juga dapat mengunggah file hingga kurang lebih 90gb.

Perlu diketahui, Web FTP PENS hanya dapat digunakan oleh mahasiswa aktif PENS yang memiliki NetID dan password terdaftar.

Cara Hosting Student Site ke Web FTP PENS

Caranya cukup mudah dan simpel, kalian cuma butuh NetID terdaftar dan koneksi internet serta file website kalian. Lalu ikuti langkah dibawah ini.

  1. Siapkan Hasil Coding

    Pastikan file website statis kalian sudah siap. Jangan lupa halaman landing page kalian harus dengan format penamaan index.html

  2. Buka Link

    Kunjungi link Web FTP PENS melalui tombol dibawah ini.

    Kunjungi

  3. Pilih Fitur

    Cari dan klik pada menu student site. Setelah itu akan diarahkan ke halaman situs EEPIS Student Portal.

  4. Pilih Fitur

    Login menggunakan NetID dan password yang telah terdaftar.

  5. Upload File

    Setelah login berhasil, upload file dengan klik salah satu tombol upload
